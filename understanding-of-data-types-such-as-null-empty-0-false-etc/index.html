之所以决定写这片笔记，是因为一直对 空 这个概念很模糊，在代码逻辑中常会遇到需要判断的时候，总是模拟两可。

常见的“空”有以下这些：

  • 整形0：0
  • 字符1：1
  • 字符空：””
  • 字符零：”0”
  • 空数组：[]
  • true
  • false
  • null

NUll 上面的那些都好理解，都是常见的，重点介绍一下NULL。

NULL 是什么？

Null是在计算机具有保留的值，可以用于指针不去引用对象，现在很多程序都会使用指针来表示条件，但是在不同的语言中，含义是不一样的。

这里我们只介绍 PHP 中的 NULL。

在 PHP 中，表示一个变量没有赋值、或者是被赋值的值为 NULL，以及被 unset 的。
