如何在 PHP 中生成二维码

原生 PHP 无法直接生成二维码，可以借助一些第三方的二维码库。这里使用 endroid/qr-code。

安装：

首先来了解一下二维码的一些基本特征

这是一张常见的二维码，由以下几部分构成：

  • ①：外框
  • ②：码眼
  • ③：码点
  • ④：logo
  • ⑤：文字

其中码眼和码点是必要的，其余部分则均是为二维码美化的而添加的，非必要。

简单的二维码生成

endroid/qr-code 提供三种不同的输出模式：

  • saveToFile()：生成二维码以图片的形式保存在本地
  • getString()：直接输出在浏览器
  • getDataUri()：输出二维码图片对应的 Base64

二维码美化

创建二维码时，除了可以放文本，还可以放入 URL、邮箱、深度链接等等。

其他问题

endroid/qr-code 似乎不支持外框、码眼、码点的自定义，具体可以查看这个 Issue。
