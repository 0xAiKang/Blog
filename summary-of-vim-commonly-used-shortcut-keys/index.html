Vim 是我在Linux 下比较常用的文本编辑器，这里整理一下常用的操作。

基本操作

  • 移动到行首：0
  • 移动到行尾：$
  • 光标移动到文件开始位置：gg
  • 光标移动到文件结束位置: shift + g
  • 删除所有内容：ggdG
  • 单行删除：dd
  • 单行复制：yy
  • 粘贴：p
  • 复制全部内容：ggyG

移动到指定行

在vim 中直接移动到指定行数，有三种方式（均是在命令行模式下输入，n 为指定的行号）：

  1. ngg/ nG
  2. :n
  3. vim +n filename
